# website

Personal blog and website, using [Hugo](https://www.gohugo.io) - a static website generator in Go.

The theme used is a fork of [typo](https://themes.gohugo.io/themes/typo/).  
I updated the base theme in order to enable the dark theme via CSS, and not JS, thanks to [tastefejl's fork](https://git.push-f.com/tastefejl/commit/?id=3cada41922abb83b5a5541e4b5efd3fc1659c483).

To install and build:

```
git clone https://github.com/k0pernicus/website && \
cd website && \
git submodule update --init
```
